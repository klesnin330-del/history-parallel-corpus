Параллельный корпус списков (XML-TEI)
«Притча о блудном сыне» (Лк 15:11–32)

1) Загрузка TEI

Загрузите XML-TEI файлы (Unicode). Выравнивание выполняется по словам (<w>).

2) Выравнивание

Выберите Master. Остальные подстроятся автоматически.

3) Разночтения

Кликните на слово для контекста (±10 слов). Используйте ◀/▶ для ручной правки.

4) Проект и экспорт

Загрузите файлы и нажмите «Автовыравнивание».
Краткая справка
  1. Загрузите XML-TEI файлы.
  2. Выберите один список как Master.
  3. Нажмите «Автовыравнивание».
  4. Кликните на слово для контекста (±10 слов). Используйте ◀/▶ для правки.
  5. Сохраните проект в JSON или экспортируйте в TEI.
