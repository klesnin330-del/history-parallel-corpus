Параллельный корпус списков (XML-TEI)
«Притча о блудном сыне» (Лк 15:11–32)

1) Загрузка TEI

Загрузите несколько XML-TEI файлов (Unicode). Выравнивание выполняется по словам (<w>).

2) Выравнивание

Выберите главный список (Master). Остальные подстроятся автоматически.

3) Разночтения

Если соответствия нет, отображается ---. Кликните на слово для контекста.

4) Проект и экспорт

Загрузите файлы и нажмите «Автовыравнивание».
Краткая справка
  1. Загрузите XML-TEI файлы.
  2. Выберите один список как Master.
  3. Нажмите «Автовыравнивание».
  4. Кликните на любое слово в таблице, чтобы увидеть контекст (±10 слов).
  5. Используйте ◀/▶ для ручной правки соответствия.
  6. Сохраните проект в JSON или экспортируйте в TEI.
