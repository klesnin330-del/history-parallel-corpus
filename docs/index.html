Параллельный корпус списков (XML‑TEI): «Притча о блудном сыне» (Лк 15:11–32)
Загрузка TEI → выбор листов → выравнивание → ручная правка → разночтения → экспорт TEI

1) Загрузка TEI

Загрузи несколько файлов XML‑TEI (Unicode). Для отображения славянских букв желательно наличие шрифтов BukyVede/Menaion Unicode на ПК.

2) Выравнивание

Сначала выбери главный список (Master) и диапазон листов у каждого списка (например, 117–119).

3) Разночтения

Типы определяются эвристически (автоматически). При ручной правке выравнивания типы пересчитываются.

4) Сохранение / продолжение

Это позволяет продолжить работу в следующем сеансе без повторной загрузки TEI.

5) Экспорт TEI

Экспорт: единый TEI-файл с сегментами и standOff/linkGrp выравнивания.

Параллельное представление

Загрузи TEI файлы, выбери Master и диапазоны листов.
Краткая справка
  1. Загрузка: нажми «Загрузка TEI» → выбери несколько XML‑TEI файлов.
  2. Master: у одного списка отметь «Master».
  3. Листы: у каждого списка укажи диапазон (например, 117–119). Это фильтрует сегменты по <pb>.
  4. Автовыравнивание: нажми «Автовыравнивание».
  5. Ручная правка: в таблице у каждой ячейки есть кнопки ▲/▼ для сдвига выравнивания по строкам.
  6. Разночтения: включай/выключай типы и режим фильтрации слева.
  7. Сохранить/продолжить: «Скачать проект (.json)», потом «Загрузить проект».
  8. Экспорт TEI: «Скачать TEI» — единый файл с параллельным корпусом.
Если таблица большая — используй поиск по master (вверху справа).
